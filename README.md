## Meu Relatório de Exercícios: Lógica de Programação e Algoritmos - Tiago Gomes Cruz

Nessa lista de exercícios, aproveitei para treinar bastante os conceitos básicos de programação e lógica com vários desafios práticos.

## Exercício 1
No primeiro código, usei o laço `for` junto com o operador de resto da divisão (`%`) para separar e mostrar os números pares e ímpares de 0 a 100.

## Exercício 2
Depois, trabalhei com listas para receber três números e descobrir rapidinho qual era o maior e o menor deles usando funções nativas.

## Exercício 3
No terceiro desafio, brinquei um pouco com strings e criei um efeito de escada, imprimindo o nome que o usuário digitou letra por letra de forma incremental.

## Exercício 4
Na sequência de Fibonacci, usei a matemática a meu favor e fui atualizando duas variáveis para calcular a soma dos termos até onde o usuário escolheu.

## Exercício 5
No quinto exercício, criei um validador de dados bem rigoroso para garantir que informações como idade, salário e estado civil só fossem aceitas se estivessem certas.

## Exercício 6
Também fiz um script para testar se um número era primo, controlando quantas vezes ele era dividido de forma exata dentro de um laço de repetição.

## Exercício 7
Para calcular o fatorial de um número, usei o laço `for` para fazer uma contagem regressiva e ir multiplicando os valores até chegar no resultado.

## Exercício 8
No oitavo exercício, explorei bastante o que dá para fazer com listas, puxando o tamanho dela, somando os itens, achando os extremos e ordenando tudo de forma crescente e decrescente.

## Exercício 9
Logo depois, montei um dicionário no esquema de chave e valor para organizar os dados da tabela que a atividade pedia.

## Exercício 10
No décimo programa, usei o laço `while` para fazer uma trava de segurança que só libera o acesso quando o usuário adivinha a senha numérica correta.

## Exercício 11
Por fim, fechei a lista criando um gerador de tabuada bem simples, onde o usuário digita um número e o programa faz a multiplicação automática de 1 a 10.
